# Contributing to SVG Pattern Generator

First off, thanks for taking the time to contribute! 🎉

The following is a set of guidelines for contributing to SVG Pattern Generator.

## How Can I Contribute?

### Reporting Bugs

- Use the GitHub issue tracker to report bugs.
- Describe the bug in detail and provide steps to reproduce it.

### Suggesting Enhancements

- Use the GitHub issue tracker to suggest new features or improvements.

### Pull Requests

1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes (`vp check`).
5. Make sure your code follows the existing style.

## Development Setup

This project uses [VitePlus](https://viteplus.dev/).

```bash
# Install dependencies
vp install

# Start development server
vp dev

# Run checks (lint, format, types)
vp check

# Build for production
vp build
```
